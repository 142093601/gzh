# 公众号文章发布前自检清单

写完文章后，用这份清单逐项检查。每一项都要过，没过的先改再发。

---

## 一、标题检查

- [ ] **字数**：是否在 12-28 字之间？（太短没信息量，太长被截断）
- [ ] **吸引力**：去掉修饰词后，核心信息是否还成立？
- [ ] **是否包含数字**？数字型标题点击率平均高 30%
- [ ] **是否有情绪词**？（震惊、扎心、真相、终于、竟然……）
- [ ] **是否避免了标题党痕迹**？别用"不转不是中国人"这类
- [ ] **搜一搜友好**：标题里是否包含用户会搜的关键词？
- [ ] **副标题/摘要**：是否填写？决定折叠后的第二眼吸引力

## 二、开头检查（前 3 行）

- [ ] **前 3 行能否制造"继续读"的冲动？** 遮住后面，只看开头问自己
- [ ] **是否使用了明确的 hook 类型？**（参考 hooks.md）
- [ ] **是否在 50 字内进入正题？** 别废话太多
- [ ] **开头和标题是否呼应？** 标题承诺的，开头要兑现
- [ ] **有没有"大家好，我是 XX"？** 除非你是大号，否则删掉

## 三、结构检查

- [ ] **小标题数量**：每 300-500 字是否有小标题分隔？
- [ ] **段落长度**：每段是否不超过 5 行？（手机屏幕上 5 行已经很长了）
- [ ] **逻辑链条**：每部分之间是否有承接关系？读起来是否通顺？
- [ ] **有没有"信息过载"？** 一篇文章讲透 1 个点，比泛泛讲 5 个点强
- [ ] **结尾是否有收束感？** 不能虎头蛇尾，也不能突然结束

## 四、互动与留存

- [ ] **全文是否有至少 1 个互动锚点？**（提问/投票引导/"你有没有遇到过"）
- [ ] **结尾是否有 CTA？**
  - 关注引导："关注 XX，获取更多 XX"
  - 转发引导："觉得有用就转给 XX 吧"
  - 评论引导："你在 XX 方面有什么经历？评论区聊聊"
- [ ] **是否有"钩子"留到下一篇？** 系列文章或预告下期内容

## 五、视觉与排版

- [ ] **配图是否到位？** 长文至少 3-5 张图，避免大段纯文字
- [ ] **配图是否有意义？** 别为了配图而配图，每张图要服务内容
- [ ] **配图是否高清无水印？** 模糊图和竞品水印会拉低质感
- [ ] **重点是否加粗？** 金句、关键结论、核心数据要视觉突出
- [ ] **引用块/分割线是否使用得当？** 区分章节，但别花里胡哨
- [ ] **全文配色是否统一？** 主色+辅色不超过 3 种

## 六、质量底线

- [ ] **错别字检查**：通读一遍，重点看易错词（的地得、在再、做作）
- [ ] **敏感词检查**：是否有平台违禁词？（可用工具辅助）
- [ ] **事实核查**：引用的数据、案例、人名是否准确？
- [ ] **版权检查**：图片是否有授权？引用是否注明出处？
- [ ] **预览检查**：在手机上预览一遍，确认排版在移动端正常

## 七、发布时机

- [ ] **发布时间**：是否选在目标读者的活跃时段？
  - 早高峰 7:00-9:00（通勤碎片时间）
  - 午休 12:00-13:00（午饭时间）
  - 晚间 20:00-22:00（黄金阅读时间）
- [ ] **是否避开了竞品集中发布时段？** 同领域大号发了，你晚 1-2 小时
- [ ] **热点时效**：如果是蹭热点，是否还在热度窗口内？

---

## 快速自检版（30 秒版）

赶时间时只查这几项：

1. 标题有数字或情绪词吗？
2. 前 3 行有 hook 吗？
3. 段落不超过 5 行吗？
4. 结尾有 CTA 吗？
5. 手机预览过吗？

5 项全过 → 可以发。
