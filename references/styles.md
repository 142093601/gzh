# 公众号风格范例库

各领域头部账号的写作风格拆解，供构思文章时参考对标。

---

## 一、科技互联网

### 半佛仙人
- **调性**：毒舌、暴躁、幽默、硬核
- **结构**：无明显分段标题，意识流叙事，一气呵成
- **特点**：
  - 用段子和比喻讲复杂的商业/金融逻辑
  - 极强的个人语感，密集的短句和反问
  - 深度拆解利益链条，"看穿本质"是核心卖点
  - 不讲道理，讲故事和博弈
- **标题套路**：反常识 + 情绪化，如"XX的真相，比你想的更离谱"
- **适合参考**：想写有态度、有观点的深度解读

### 差评
- **调性**：犀利吐槽、接地气、有正义感
- **结构**：热点引入 → 事件还原 → 深度分析 → 观点输出
- **特点**：
  - 擅长拆穿互联网套路和消费陷阱
  - 大量截图和证据，"有图有真相"
  - 语言通俗，不端着，像朋友聊天
- **标题套路**：悬念 + 吐槽，如"XX，你在逗我？"
- **适合参考**：想写消费者视角的评测/吐槽文章

### 极客公园
- **调性**：专业、理性、前瞻
- **结构**：新闻引入 → 背景铺垫 → 产品/技术分析 → 行业影响
- **特点**：
  - 信息密度高，专业术语但解释到位
  - 善用数据和图表支撑观点
  - 视角偏产业端，关注趋势和创新
- **标题套路**：直接 + 具体，如"XX发布XX，意味着什么"
- **适合参考**：想写专业但不枯燥的行业分析

## 二、情感/生活方式

### 新世相
- **调性**：文艺、温暖、有共鸣感
- **结构**：故事开头（通常是普通人的故事） → 情感共鸣 → 观点升华
- **特点**：
  - 擅长捕捉都市人的细腻情绪
  - 文字克制但有力量，不煽情
  - 经常发起话题征集，UGC 内容占比高
  - 善用第一人称叙事
- **标题套路**：情绪 + 场景，如"成年人的崩溃，都是静音的"
- **适合参考**：想写引发情感共鸣的走心文章

### 十点读书
- **调性**：温暖治愈、正能量、家庭向
- **结构**：故事/案例引入 → 道理拆解 → 金句总结 → 互动引导
- **特点**：
  - 大量引用名人名言和书籍观点
  - 内容偏鸡汤但不空洞，有实际建议
  - 配音/音频是一大特色
  - 受众偏女性、30+
- **标题套路**：数字 + 情感，如"人到中年，最好的活法就一个字"
- **适合参考**：想写治愈向、家庭向的内容

### 夜听
- **调性**：深夜电台感、陪伴感、温柔
- **结构**：短文 + 音频，文字简练，留白多
- **特点**：
  - 每篇很短（500-800字），重情绪轻逻辑
  - 第二人称"你"，制造一对一陪伴感
  - 配合音频，适合睡前场景
- **适合参考**：想写短小精悍的陪伴型内容

## 三、职场/成长

### 插座学院
- **调性**：实用、干货、直给
- **结构**：痛点引入 → 方法论拆解 → 案例佐证 → 行动清单
- **特点**：
  - 内容高度结构化，标题就是观点
  - 大量职场案例和真实故事
  - 语言简洁，不绕弯子
  - 强调"可操作性"
- **标题套路**：痛点 + 解法，如"工作5年还没升职？你可能犯了这3个错"
- **适合参考**：想写实用的职场方法论文章

### 粥左罗
- **调性**：励志、方法论、个人品牌向
- **结构**：个人经历切入 → 总结方法 → 鼓励行动
- **特点**：
  - 大量使用自己的成长故事做案例
  - 强调"刻意练习"和"复利思维"
  - 语言积极向上，有煽动性
  - 善用数字标题和清单体
- **标题套路**：数字 + 结果导向，如"我是如何用1年从月薪5千到年入百万的"
- **适合参考**：想写个人成长、逆袭故事

## 四、创意/脑洞型

### GQ实验室
- **调性**：脑洞大开、讽刺幽默、年轻化
- **结构**：非常规——长图漫画、互动测试、虚拟对话、清单体等
- **特点**：
  - "去杂志化"，摆脱传统媒体严肃感
  - 形式创新大于文字本身（漫画、长图、互动）
  - 擅长用反讽解构社会现象
  - 广告内容化做得最好，"广告即内容"
- **标题套路**：反差 + 好奇，如"如果你的XX会发朋友圈"
- **适合参考**：想突破传统图文形式，做创意内容

### 局部气候调查组
- **调性**：知识科普 + 视觉创意
- **结构**：超长信息图，一图流叙事
- **特点**：
  - 用设计讲故事，文字极少
  - 选题冷门但有趣
  - 视觉冲击力强
- **适合参考**：想做视觉驱动的科普内容

## 五、财经/商业

### 吴晓波频道
- **调性**：理性分析、中产视角、有纵深感
- **结构**：热点切入 → 历史/数据铺垫 → 趋势判断
- **特点**：
  - 善于把当下事件放入历史坐标系
  - 语言成熟稳重，知识分子腔调
  - 大量引用经济数据和研究报告
- **适合参考**：想写有深度的商业/经济分析

## 使用指南

1. **选题阶段**：根据你要写的文章类型，找到 1-2 个对标账号
2. **大纲阶段**：参考对标账号的常用结构，但不要照搬
3. **写作阶段**：参考对标账号的语言节奏和情绪处理方式
4. **标题阶段**：收集对标账号近期 10 篇文章的标题，提炼套路

> 注意：参考风格 ≠ 抄袭。学习的是结构、节奏和手法，内容必须是原创的。
